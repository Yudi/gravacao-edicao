Capturando áudio com o OBS e iShowU Audio Capture

Para que o OBS posa gravar o áudio do computador no macOS, é necessário instalar um segundo programa, o iShowU Audio Capture.

Para baixar o iShowU Audio Capture, selecione um dos botões de download abaixo, de acordo com a versão do seu macOS.

Não tem certeza sobre qual versão você está utilizando? Clique aqui para saber como descobrir a versão do seu macOS.


Após o download, abra o arquivo .dmg e execute o arquivo .pkg que está janela que será aberta. Atentamente, leia as instruções na tela.

Clique nas imagens para ampliá-las.

Após a instalação, pode ser que o programa precise de uma permissão de segurança para poder funcionar.
É possível conceder esta permissão em Preferências do sistema → Segurança e Privacidade → Geral. Quando for solicitado, NÃO reinicie o computador. Acompanhe as imagens:

1. Aviso de extensão bloqueada

2. Preferências do sistema


3. Segurança e privacidade

4. Não reinicie o computador


Espere a instalação ser finalizada e reinicie o seu computador logo após.



Configuração de áudio

Pesquise por Configuração de Áudio e MIDI no Spotlight (pressione ⌘+Espaço para abri-lo).

Em Configuração de Áudio e MIDI, clique no botão de adicionar novo dispositivo (+) e, depois, em Criar Dispositivo com Saída Múltipla


No Dispositivo com Saída Múltipla, confira se o Dispositivo Principal são os Alto-falantes do computador, depois, marque as caixinhas dos Alto-falantes do computador e do iShowU Audio Capture.

Vamos mudar o dispositivo de saída de áudio do computador: Preferências do sistema → Som → Saída.

1. Preferências do sistema

2. Selecione o Dispositivo com Saída Múltipla e marque a caixa Mostrar volume na barra de menus


Quando o Dispositivo com Saída Múltipla está selecionado, não é possível ajustar o volume do sistema. Mas, com a caixinha que marcamos anteriormente, fica mais fácil para alternar entre os dispositivos de saída. Note que há um novo ícone na barra na parte superior da tela. Ao clicar neste ícone, é possível selecionar os dispositivos de áudio:


Para ajustar o volume, basta selecionar os Alto-falantes do computador.
Lembre-se de que o OBS não gravará áudio se o Dispositivo com Saída Múltipla não estiver selecionado.
